## 1. Краткое описание проекта

Разработана интеллектуальная система автоматизации для управляющей компании складского комплекса. Проект объединяет три направления работы с клиентами в единый AI-контур:

| Контур | Назначение | Результат |
|--------|-----------|-----------|
| 🛒 **Sales** | Консультации по аренде, сбор лидов 24/7 | Рост конверсии, работа без выходных |
| 🛠 **Support** | Приём заявок от арендаторов, ИИ-классификация | Разгрузка диспетчерской на ~60% |
| 📢 **Communications** | Генерация постов, авто-уведомления | Экономия 10+ часов/неделю на маркетинге |

---

## 2. Что получилось реализовать ✅

### 🔹 Техническая реализация

| Компонент | Статус | Описание |
|-----------|--------|----------|
| **Язык и фреймворки** | ✅ Готово | Python 3.12, FastAPI, Aiogram 3.x |
| **Интеграция с ИИ** | ✅ Готово | Подключение YandexGPT API для генерации ответов и классификации заявок |
| **Telegram-бот** | ✅ Готово | Приём заявок, управление статусами, генерация постов по команде `/post` |
| **Веб-виджет** | ✅ Готово | Диалоговый интерфейс для сайта на Tilda с валидацией данных |
| **База данных** | ✅ Готово | SQLite + SQLAlchemy для хранения заявок, лидов и истории диалогов |
| **База знаний** | ✅ Готово | Вынесена в `knowledge_base.py` — обновление без правки кода |
| **Деплой** | ✅ Готово | Ubuntu VPS, Nginx (reverse proxy), systemd (автозапуск), Certbot (HTTPS) |
| **Безопасность** | ✅ Готово | Конфигурация через `.env`, CORS, rate-limiting, защита от спама |

### 🔹 Функциональные возможности

#### 🤖 Telegram-бот:
- Приём заявок от арендаторов с автоматическим извлечением: категория, приоритет, номер офиса
- ИИ-классификация заявок: `электрика` / `сантехника` / `техническое` / `другое`
- Мгновенные уведомления админам в группу с карточкой заявки
- Управление статусами: `new` → `in_progress` → `done` с историей изменений
- Генерация постов для канала по команде `/post` + вводные от админа
- Поиск и фильтрация заявок по статусу, дате, категории

#### 🌐 Виджет для сайта (Tilda):
- Диалоговый интерфейс: посетители пишут в чат → получают ответ от ИИ
- Интеграция с базой знаний для точных ответов по ценам и условиям
- Валидация телефона, защита от некорректного ввода и галлюцинаций ИИ
- Сбор лидов с сайта → отправка в ту же админ-группу Telegram
- Защита: CORS, rate-limiting, защита от спама

#### 🗄 Общие компоненты:
- Модульная архитектура: `llm.py`, `models.py`, `database.py` — легко масштабировать
- Логирование через `loguru` с ротацией для удобной отладки
- Конфигурация через `.env` — ключи не хранятся в коде
